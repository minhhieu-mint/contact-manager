# Contact Manager

> A simple command-line contact manager written in Python.

---

## 📖 Introduction

Contact Manager is a simple command-line application that allows users to manage personal contacts. The application stores all contact information in a JSON file and provides essential features such as viewing, adding, editing, deleting, searching, and clearing contacts.

This project was developed as a personal learning project to practice Python fundamentals and improve programming skills. During development, I learned how to organize code into functions, validate user input, work with JSON files, and use Git and GitHub for version control.

---

## ✨ Features

- View all contacts
- Add a new contact
- Edit existing contacts
- Delete contacts
- Search contacts by name
- Search contacts by phone number
- Search contacts by email
- Search contacts across all fields
- Clear all contacts

## ▶️ How to Run

Clone the repository to your computer.

```bash
git clone https://github.com/minhhieu-mint/contact-manager
```

Move into the project directory.

```bash
cd ContactManager
```

Run the program.

```bash
python main.py
```
